#### 线性结构的特点

```
前提： 数据元素的非空有限集合内
```

1. 存在唯一的一个被称为第一个的数据元素
2. 存在唯一的一个被称为最后一个的数据元素
3. 除第一个以外，集合中的每个数据元素都有一个前驱
4. 除最后一个以外，每一个数据元素都有一个后继
